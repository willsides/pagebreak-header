Displays header text within a pagebreak line

== Installation ==

1. Upload the following files and directories to the `/wp-content/plugins/pagebreak-header` directory:
	* build/
	* block.json
	* pagebreak-header.php

2. Activate the plugin through the 'Plugins' screen in WordPress
